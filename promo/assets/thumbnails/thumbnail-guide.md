# YouTube Thumbnail Guide

## Specs
- **Size:** 1280x720 pixels (16:9)
- **Max file size:** 2 MB
- **Format:** JPG or PNG

## Design Principles
- **Large text:** 3-5 words maximum — readable at mobile thumbnail size
- **High contrast:** Light text on dark background or vice versa
- **Dashboard screenshot:** Show the actual TradeMetrics Pro interface — authenticity converts
- **Brand accent color:** Use TradeMetrics Pro green (#00ff9d) for text highlights and borders
- **No clutter:** One focal point, one message

## Tools
- **Canva** (recommended) — Free tier works fine, drag-and-drop, 1280x720 preset available
- **GIMP** — Free, more control, steeper learning curve

## Brand Consistency
- Always use the same font for text overlays (Inter Bold or Montserrat Bold)
- Always include the green accent (#00ff9d) somewhere — border, text highlight, or glow
- Keep the dark background consistent across all thumbnails for recognizable series branding
